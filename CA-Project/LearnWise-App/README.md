# Project Description

LearnWise is a comprehensive learning platform designed to support students throughout their academic journey. It serves as a centralized hub for managing assignments, tracking grades, and accessing educational resources.

## Key Features

- **Assignment Management**: Create, organize, and track all assignments in one place.
- **Grade Tracking**: Monitor academic performance with detailed grade records.
- **Resource Hub**: Access and manage learning materials and notes.
- **User-Friendly Interface**: A clean and intuitive design for seamless navigation.

## Technologies Used

- **Framework**: Flutter
- **Language**: Dart
- **Platform**: Android / Web / Desktop
- **Build System**: Gradle
